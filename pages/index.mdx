# Book of Scotland Intranet

## DIRECTORIES

The directories are constructed with [Django](http://34.244.159.150:8000/admin/) components ('apps'). These use [Modular Script construction tools](http://34.244.159.150:8000/admin/modular_scripts_app/) which draw on [Backend](http://34.244.159.150:8000/admin/backend/) script construction elements

Directories are mostly expanded [manually](http://34.244.159.150:3000/HowTo/ExpandDirectory/Expand_Manually) so far, but this will later be mostly [automated](http://34.244.159.150:3000/HowTo/ExpandDirectory/Automated_Expansion).
